## 单词卡片重新设计方案

### 核心改进

1. **尺寸与留白优化**
   - 卡片最小高度：手机端 420px，桌面端 480px
   - 内边距：手机端 p-10，桌面端 p-12
   - 更宽敞的竖向比例，适合单词展示

2. **视觉设计升级**
   - 渐变背景：从白色到淡紫色/淡蓝色的柔和渐变
   - 玻璃拟态效果：backdrop-blur + 半透明边框
   - 多层阴影：营造立体感和高级感
   - 微交互动画：悬停时卡片轻微上浮 (translateY -4px)

3. **排版优化**
   - 英文单词：text-5xl sm:text-6xl，font-extrabold，tracking-tight
   - 音标：斜体等宽字体，text-indigo-500/70
   - 中文释义：text-2xl，leading-relaxed，max-width 限制
   - 分隔线：渐变线条 (from-transparent via-slate-300 to-transparent)

4. **状态标识改进**
   - Mastered：渐变 emerald 背景，带发光效果
   - Reviewing：渐变 amber 背景，带发光效果
   - 添加平滑过渡动画

5. **按钮设计**
   - Mastered 按钮：渐变 indigo 背景 + 阴影 + 悬停发光
   - Review Later 按钮：幽灵按钮风格，hover 时背景填充
   - 统一圆角 rounded-2xl

### 技术实现
- 修改 StudyMode/index.tsx 组件
- 使用 Tailwind CSS 实现所有效果
- 保持翻转功能 intact
- 确保响应式设计
