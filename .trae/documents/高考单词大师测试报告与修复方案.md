## 测试发现的关键问题

### 🔴 P0 优先级：WordList 滚动加载不触发

**问题描述**：用户报告在 Full Dictionary 页面只能看到 abstract 为止，后续单词无法加载。

**根本原因**：
1. 滚动触发阈值 `scrollHeight - 100` 可能不够精确
2. 容器高度计算在某些情况下不准确
3. 缺少防抖处理可能导致多次触发或触发失败

**修复方案**：
```typescript
// 1. 调整触发阈值
const handleScroll = useCallback((e: React.UIEvent<HTMLDivElement>) => {
  const { scrollTop, scrollHeight, clientHeight } = e.currentTarget;
  const scrollBottom = scrollTop + clientHeight;
  const threshold = scrollHeight - 50; // 从100改为50，更容易触发
  
  if (scrollBottom >= threshold && hasMore) {
    setDisplayCount(prev => Math.min(prev + ITEMS_PER_PAGE, filteredWords.length));
  }
}, [hasMore, filteredWords.length]);

// 2. 确保容器有明确的高度
// 在父组件中设置固定高度或最小高度
```
